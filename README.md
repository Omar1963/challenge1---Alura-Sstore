# Alura Store — Análisis de Datos Multitienda

Este proyecto forma parte del **Challenge de Alura LATAM**, donde se analizan los datos de cuatro tiendas diferentes para obtener insights comerciales, operativos y de comportamiento de clientes.  
El análisis se desarrolla utilizando **Python y Pandas**, integrando múltiples fuentes de datos y generando visualizaciones claras para la toma de decisiones.


## Objetivo del análisis

El propósito principal del proyecto es evaluar el desempeño de las tiendas de Alura Store a partir de cinco dimensiones clave:

1. **Facturación total por tienda**  
   Determinar cuál tienda genera mayores ingresos.

2. **Categorías más populares por tienda**  
   Identificar qué tipos de productos se venden más en cada tienda.

3. **Promedio de evaluación por cliente**  
   Analizar la satisfacción general de los clientes.

4. **Productos más y menos vendidos por tienda**  
   Detectar oportunidades de mejora y productos estrella.

5. **Costo promedio de envío por tienda**  
   Evaluar la eficiencia logística y costos asociados.

### Tienda lider  
La tienda líder no solo vende más, sino que lo hace de manera consistente. Su catálogo y ticket promedio la posicionan como la referencia del ecosistema.”

### Categorías más vendidas
Tecnología domina el escenario. Los clientes buscan innovación, y las tiendas que ofrecen más variedad en esta categoría capturan la mayor parte del mercado.”

### Evaluación promedio por cliente
Aunque la mayoría de los clientes están satisfechos, existe un pequeño grupo con experiencias negativas que podría estar afectando la reputación general.”

### Productos más y menos vendidos
Los productos estrella impulsan la facturación, pero los productos de baja rotación representan un costo silencioso que debe gestionarse.”

### Costo promedio de envío

Las diferencias entre tiendas sugieren que algunas rutas o proveedores podrían optimizarse para reducir costos sin afectar la experiencia del cliente.”

## Conclusiones ejecutivas

1. La tienda con mayor facturación debe ser tomada como referencia  
Su combinación de catálogo, precios y logística es la más eficiente.

2. Las categorías tecnológicas impulsan el crecimiento  
Invertir en stock, variedad y promociones en esta categoría puede aumentar la facturación global.

3. La satisfacción del cliente es buena, pero no excelente  
Un pequeño grupo de clientes insatisfechos podría estar afectando la percepción general.  
Se recomienda revisar casos individuales.

4. Existen productos con baja rotación  
Estos productos generan costos de almacenamiento.  
Se recomienda:  
- Promociones  
- Bundles  
- Descontinuación selectiva

5. El costo de envío varía demasiado entre tiendas  
Optimizar rutas y proveedores podría reducir costos sin afectar la experiencia del cliente.
